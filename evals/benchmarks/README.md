# Benchmarks

Capture per-run benchmark summaries and metric exports here.

Recommended QEMU guest-bridge metrics:
- bridge startup success rate
- time to `runtime_ready`
- time to `desktop_ready`
- time to `bridge_attached`
- viewer-only / bridge-waiting fallback rate
- action success rate by family (observation, screenshot, shell/filesystem, desktop, browser)
- Taskers dogfood success rate
- bridge failure classification rate
- Xvfb regression success rate
